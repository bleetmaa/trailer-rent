# Trailer Rent - Strato DNS Setup Guide
# Location: C:\Windows\System32\drivers\etc\hosts
# (Run Notepad as Administrator to edit)

# STEP 1: Get your public IP
# Visit: https://whatismyipaddress.com/
# Example result: 85.123.45.67

# STEP 2: Configure Strato DNS Records
# Go to: Webbserver → DNS → DNS-inställningar
# Add these A records:
# Type: A, Name: @, Value: YOUR_PUBLIC_IP, TTL: 3600
# Type: A, Name: dev, Value: YOUR_PUBLIC_IP, TTL: 3600  
# Type: A, Name: qa, Value: YOUR_PUBLIC_IP, TTL: 3600
# Type: A, Name: www, Value: YOUR_PUBLIC_IP, TTL: 3600

# STEP 3: Add to Windows hosts file (for immediate testing)
# Replace YOUR_PUBLIC_IP below with your actual IP from Step 1

YOUR_PUBLIC_IP trailerrent.se
YOUR_PUBLIC_IP dev.trailerrent.se  
YOUR_PUBLIC_IP qa.trailerrent.se
YOUR_PUBLIC_IP www.trailerrent.se

# STEP 4: Router Port Forwarding Required
# Port 80 → 192.168.1.66:31720
# Port 443 → 192.168.1.66:31915

# STEP 5: Test DNS (after 15-30 minutes)
# Command: nslookup trailerrent.se
# Should return: YOUR_PUBLIC_IP

# STEP 6: Remove local entries once DNS works
# (Comment out or delete the YOUR_PUBLIC_IP lines above)
